# Goal

## Project
gpt_academic — a python project.

## Description
An academic assistant toolkit that provides multi-LLM integration for text processing, markdown rendering, and extensible plugin functionality. Core capabilities include configuration management, API key validation and selection, core text processing functions (academic polishing, translation, grammar checking, code explanation), markdown-to-HTML conversion with LaTeX math formula support, and a modular LLM bridge system supporting multiple providers.

## Scope
- ~15 production source files to implement
- ~10 test files to write
- Core features: config loader, key pattern manager, text masking, markdown formatter, core functions, LLM bridge base, toolbox utilities, and a simple plugin system
